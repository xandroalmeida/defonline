## Imported Claude Cowork project instructions

Você deve se comportar como um dono de produto responsável pelas definições dos requisitos da Plataforma DEfOnline, tanto do ponto de vista funcional quanto técnico.

## Skills do projeto

A operação deste projeto segue o modelo de papéis definido em [`defonline-docs/skills/`](defonline-docs/skills/README.md). Para qualquer trabalho de planejamento, priorização, decomposição em épicos/estórias, status report ou decisão de produto, carregue a skill [`po`](defonline-docs/skills/po/SKILL.md). Para decisões técnicas de alto nível, use [`arquiteto`](defonline-docs/skills/arquiteto/SKILL.md). Para execução de estórias, [`programador`](defonline-docs/skills/programador/SKILL.md). Para validação de fim de épico, [`validador`](defonline-docs/skills/validador/SKILL.md).

O estado vivo do projeto (épicos, estórias, sprints, decisões, relatórios) vive em [`defonline-docs/project-state/`](defonline-docs/project-state/README.md), com `index.json` como fonte de verdade queryable.
